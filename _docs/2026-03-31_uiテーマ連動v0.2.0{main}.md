# 2026-03-31 uiテーマ連動v0.2.0{main}

## 目的
- `--ui-theme` と `/kobold-lite` テーマ切替（light/dark/classic）を完全連動し、KoboldCpp風の見た目同等化を行う。

## 変更点
- CLI:
  - `hypura serve --ui-theme <light|dark|classic>` をサーバー初期テーマへ反映。
- API:
  - `GET /api/extra/ui-theme`
  - `POST /api/extra/ui-theme`
- GUI:
  - テーマセレクタ追加。
  - CSS変数による3テーマ切替を実装。
  - `localStorage` とサーバー設定値を同期。
- 既存UI拡張（Mode / Model / Presets / History / Event）に全テーマ適用。
