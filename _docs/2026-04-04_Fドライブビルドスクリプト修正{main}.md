# 実装ログ: build-hypura-hf.ps1 修正と F ドライブビルド手順

## 変更

1. **文字コード**: UTF-8 BOM + 実行時メッセージを英語にし、Windows PowerShell 5.1 の誤解析（文字化けで文字列未閉じ）を防止。
2. **ドライブルート**: `Drive = 'F:\'`（単引用符）で終端バックスラッシュ問題を回避。
3. **PATH**: `~\.cargo\bin` を先頭に追加（非対話シェルで `cargo` が見つからない場合の対策）。

## エージェント環境での制約

- リモート実行環境に **cargo が未インストール**のため、ここではビルド完了まで実行不可。
- ユーザーの F: 空き **13.74 GiB** は既定 **15 GiB** 未満のため、ローカルでは `-MinFreeGiB 12` などで通すか、F: を空ける。

## ローカルで実行するコマンド（リポジトリルート）

```powershell
.\scripts\build-hypura-hf.ps1 -MinFreeGiB 12
```

（15 GiB 以上空いていれば `-MinFreeGiB` は不要。）

成功時: `F:\hypura-cargo-target\release\Hypura.exe` をビルドし、`target\release\Hypura.exe`（と `hypura.exe`）へ上書きコピー。
